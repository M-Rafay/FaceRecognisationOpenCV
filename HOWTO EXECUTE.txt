PYTHON 3.6
NUMPY
RECOGNIZER = LBPH FACE RECOGNIZER

STEPS TO RUN THE PROGRAM:

1.FIRST GENERATE DATASET FOR YOUR FACE BY RUNNING
dataSetGenerator.py
2.SECOND TRAIN YOUR DATASET BYRUNNING trainer.py. IT CREATES
AN XML AND YML FILE FOR THE DATA SET
3.AT LAST RUN THE detector.py TO USE FACE RECOGNIZATION

(ONCE YOU HAVE RUN dataSetGenerator.py AND trainer.py FOR YOUR FACE 
YOU HAVE NO NEED TO RUN THAT AGAIN AND AGAIN)
